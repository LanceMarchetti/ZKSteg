CoverKey Prototype

Raw-byte host-referenced coordinate encoding. The host file is not modified. The generated .ckey file stores packed coordinates pointing into the host bytes.

Prototype note: Test with small files first. Large files may temporarily freeze the browser. This version uses no external libraries and stores the key as a compact binary .ckey file, not JSON and not Base64.
